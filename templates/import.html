如何导出浏览器书签？

Chrome / Edge

  1. 打开浏览器，按 Ctrl+Shift+O 打开书签管理器
  2. 点击右上角的 ⋮ 菜单
  3. 选择「导出书签」
  4. 保存为 HTML 文件

Firefox

  1. 按 Ctrl+Shift+O 打开书签管理器
  2. 点击「导入和备份」
  3. 选择「将书签导出为 HTML」
  4. 保存文件
